Process

Beginning the “tear down” with a Dell system unit.

Take off the side panel.

Take off the CPU Case with a screwdriver.

Open the CPU lock and take out the CPU.

Unplug the data cable connecting to the DVD drive.

Unplug the P4 cable connecting to the DVD drive.

Unplug the P3 cable connecting to the motherboard.

Unplug the power cable connecting to the hard drive.

Unplug the main power cable for the motherboard.

Unscrew and take out the power supply.

Take out the two RAMs.

Unplug fan cables.

Unscrew screws on the motherboard and take the motherboard out.

Case without the motherboard.

Motherboard outside the case.

Take out the I/O side panel.

Take out the floppy disk drive.

Take out the hard drive.

RAM ouside the motherboard.

CPU ouside the motherboard.

Hard disk drive outside the system unit.

Floppy disk drive outside the system unit.

Fan outside the system unit.

All components outside the case.

Empty case.

Install the hard disk drive in the system unit.

Install the motherboard in the system unit.

Install the I/O ports in the system unit.

Install the heat sink stand in the system unit.

Install the anti theft alarm in the system unit.

Install the fan in the system unit.

Stablize the motherboard with screws.

Connect the SATA cable to the hard drive.

Install the two RAMs into the motherboard.

Connect all power and data cables.

Install and lock the CPU.

Install the heat sink with a screwdriver.

Install the DVD drive.

Finished installation inside the system unit.

Install the side panel of the case.

Back view of the system unit.
